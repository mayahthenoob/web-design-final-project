✅ How to Fix “SQLSTATE[HY000] [2002] No such file or directory” on InfinityFree
InfinityFree does NOT use localhost for MySQL connections.

✔️ Step 1: Find Your MySQL Hostname
Go to your InfinityFree Control Panel (vPanel)

Open MySQL Databases

Look for MySQL Hostname — it looks like:

sqlXXX.epizy.com
or

sqlXXX.infinityfree.com
✔️ Step 2: Update your PHP database settings
If your code looks like this:

$dsn = "mysql:host=localhost;dbname=yourdbname";
$username = "yourusername";
$password = "yourpassword";
Change host=localhost to the hostname given in vPanel:

$dsn = "mysql:host=sqlXXX.epizy.com;dbname=yourdbname";
$username = "yourusername";
$password = "yourpassword";
✔️ Other things to check
1. Ensure the database name is correct
InfinityFree DB names look like:

epiz_12345678_dbname
2. Ensure the DB user is correct
Should look like:

epiz_12345678
3. Ensure remote MySQL is NOT used
InfinityFree does not allow external connections.
The connection must come from your hosted PHP files only.

✔️ Example working PDO connection for InfinityFree
$host = "sqlXXX.epizy.com";   // your MySQL Hostname
$dbname = "epiz_12345678_dbname";
$username = "epiz_12345678";
$password = "yourpassword";

try {
    $dsn = "mysql:host=$host;dbname=$dbname;charset=utf8mb4";
    $pdo = new PDO($dsn, $username, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connected successfully!";
} catch (PDOException $e) {
    echo "Connection failed: " . $e->getMessage();
}
